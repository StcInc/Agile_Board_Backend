# Agile Board Backend

Minimal ```nodejs / express / redis``` http api for

```
https://github.com/lkmfwe/AgileBoardFrontend
```

#### To start up:
1. Run
```bash
cp ./config_example.json ./config.json
```
2. Change the values in ```config.json``` (Optional)
3. Run
```bash
npm install
```
4. Run
```bash
node server.js
```
